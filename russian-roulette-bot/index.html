Russian Roulette Bot 1667 is an interactive Facebook bot that allows users to play a game of Russian Roulette.

Users interact with the bot via commenting on it's most recent post, the bot will periodically fetch the comment from the post and, if the user has not already been selected from the post, they will play a game of Russian Roulette.
Initially the "gun" starts fully loaded with six bullets and the state of the gun is saved between user selections, meaning that the first comment chosen has a 6/6 chance of getting "shot", the next user has a 5/6 chance, and so on. The results of the game are posted to the user in the form of a reply to their original comment, and these results are also stored in a separate data file for later use, such as the Weekly and All-Time Leaderboards.
